# A4 圆形贴纸排版工具

一款运行在浏览器中的 A4 圆形贴纸排版设计工具，采用极简现代主义风格，帮助用户快速完成圆点贴纸的排版、设计、预览与导出。

## 功能特性

### 图片处理

- **批量上传** — 支持最多上传 100 张图片，拖拽或点击区域即可完成
- **圆形裁剪** — 内建圆形裁剪器，可拖拽调整图片位置、鼠标滚轮缩放（0.01x ~ 10x），所见即所得
- **待选用区** — 裁剪完成的图片进入待选用区，支持选中、删除，圆形缩略图预览

### 背景色设置

- **CMYK 色板** — 为每个贴纸设置独立底色，C / M / Y / K 四个通道滑块 + 数值输入精确控制
- **1mm 出血** — 底色超出贴纸直径 1mm，满足印刷出血要求
- **屏幕吸色** — 支持 EyeDropper API，可直接吸取屏幕上任意颜色并转换为 CMYK 数值

### 页面排版

- **三种预设网格样式**
  - 样式一（10mm）：24 行 × 17 列，直径 10mm
  - 样式二（16mm）：16 行 × 11 列，直径 16mm
  - 样式三（20mm）：12 行 × 9 列，直径 20mm
- **自定义样式** — 自由设置页面宽高（可脱离 A4）、上下左右边距、圆点直径、行数列数，系统自动计算横向与纵向间距并实时预览
- **多页面支持** — 最多 100 页，右侧缩略图面板可添加、删除、切换页面
- **整体微调** — 支持 X / Y 方向偏移调整，精度 0.1mm；配备可视化预览框和方向键按钮

### 填充与编辑

- **多种填充方式**
  - 点击单个圆形区域直接填充
  - 工具栏「填一个」— 按行优先自动填充下一个空位
  - 工具栏「填一行」— 填充当前页第一个未满行
  - 工具栏「填满整页」— 一键填满当前页面所有空位
- **删除操作** — 「删一个」删除最后一个贴纸、「删一行」删除最后有内容的行、「删除整页」清空当前页
- **框选功能** — 开启框选填充或框选删除模式，在画布拖拽即可批量操作选中区域内的贴纸
- **撤销 / 重做** — 支持最近 20 次操作历史，可撤销和重做

### 导出

- **多格式导出** — 支持 PDF（直接导出单文件）、SVG / PNG / JPG（单页单文件或多页自动打包 ZIP）
- **300 DPI** — 栅格格式（PNG / JPG）以 300 DPI 高分辨率渲染，满足印刷需求
- **CMYK 底色保留** — PDF 导出中底色以实际 CMYK 十六进制色值呈现

### 画布操作

- **缩放** — 右下角控制面板支持 20% ~ 300% 缩放，可点击加减号或直接输入百分比，也支持 Ctrl + 滚轮缩放
- **实时预览** — 所有操作在模拟 A4 纸张的画布上实时呈现

## 技术实现

- 纯前端 HTML / CSS / JavaScript，无需构建工具或后端服务
- 依赖库（CDN 加载）：
  - [jsPDF](https://github.com/parallax/jsPDF) — PDF 生成
  - [JSZip](https://stuk.github.io/jszip/) — 多页导出打包
  - [FileSaver.js](https://github.com/eligrey/FileSaver.js/) — 浏览器端文件下载
- CMYK ↔ RGB 双向颜色转换
- Canvas 圆形裁剪与合成
- 基于 SVG 的矢量导出

## 使用方式

直接在浏览器中打开 `index.html`，无需安装或构建。所有处理均在本地完成，图片不会上传至任何服务器。
